# trpc testing

## Project setup

```bash
$ pnpm install
```

## Compile and run the project

```bash
# development
$ pnpm run dev
$ pnpm run dev:backend
$ pnpm run dev:frontend

# build mode
$ pnpm run build
$ pnpm run build:backend
$ pnpm run build:frontend

# production mode
$ pnpm run prod
$ pnpm run prod:backend
$ pnpm run prod:frontend

```
